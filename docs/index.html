Desktop TV Remote turns your computer into universal remote control for TV and other devices. This application allows you to control over 10000 devices from your computer using keyboard shortcuts or graphical interface. You can set keyboard shortcuts for any button on remote control and use them anywhere on your computer while browsing the Internet or even if you are playing some games.

Remember that this application will work only on devices that have a built-in infrared blaster.
